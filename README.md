#Магазин товаров
Это вторая версия проекта- тестового задания для ecom.tech, представляющего собой одностраничное приложение интернет-магазина с поиском товаров и модальным окном просмотра подробной информации о товаре. Проект можно посмотреть локально или на GitHubPages.

##Реализуемый стек:
- React 19
- TypeScript
- инструмент сборки веб-проектов Vite
- кастомные библиотеки React
- CSS

## Что изменилось:
1. Появилась типизация данных благодаря TS. Основной тип Item вынесен в отдельный файл types/index.ts для удобного вызова переменной во всем проекте.
2. Проект разделился на компоненты по функциям (Filter - фильтрация товаров на основе поискового запроса , Header - «шапка» сайта, ModalPortal -«рамка» для модального окна(для корректного отображения стилей и наложения), ModalWindow - модальное окно товара, ProductCard - карточка товара на основой странице в сетка, ProductGrid- сетка из товаров на главной странице), которые собираются воедино в корневом приложении App.tsx.
3. Вместе с разделением проекта пришло также разбиение index.html из прошлой версии на логические части. Теперь в каждом компоненте проекта есть функциональный компонент(React.FC), который возвращает tsx структуру, содержащую разметку соответствующего элемента.
4. Вместо addEventListener теперь используются «состояния» (states, {useState}), которые отслеживают изменения в DOM структуре.
5. 5. Проект приобрел структуру приложения React с компонентной архитектурой и встроенными инструментами разработки, в то время как первая версия представляла собой просто разрозненные файлы HTML, CSS, JS.

## Что осталось прежним:
1. CSS стили перенесены практически без изменений
2. Хранение данных в json и сами данные остались без изменений
3. HTML разметка превратилась в TSX, но структура осталась прежней.
